Aragonite

A free lightweight and onobstrusive field validator that helps you validate any kind of data before submiting to your server.

Realtimevalidation feedback

This example shows a basic implementation. While the user types, Aragonite runs a sanitation process on the client side to make sure all the fields ar filled correctly.

If the field is invalid it will add a class called ".required" to the HTML element. You customize that class and comunicate to the user that something is wrong with their inputs.

By the time the user finished filling the form, if there is a field with the class required, it will return false preventing the user from submiting wrong data or data-types. 


<form id="demo-form" class="how-form" action="#">
        <input type="text" data-type="email" placeholder="Type an Email address" />
    <input type="text" data-type="string" placeholder="Type any string" />
    <input type="text" data-type="phone" placeholder="Type a phone number" />
</form>



.required{
    border:thin solid @light-red !important; 
    background:#FCF0F0 !important;
}



<script type="text/javascript">
    aragonite.init('demo-form');
</script>

FAQ

Can i use it for free?

Yes! Aragonite is an opensource project authored by Leo Javier and released under an MIT license.

Can i ask for a new functionality?

Yes, you can submit a ticket through Github at any time. You are welcome to contact me via Email or Twitter for suggestions.

What's new in latest release?

Please take a look of the Changelog.

need to validate a field but the pattern doesn’t exist in the regex list?

Please post a question on stackoverflow and tag it as #aragonite. The right answer will be added to the Regex List so anyone including yourself can use it again in the future.
